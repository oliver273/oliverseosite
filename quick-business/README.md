## Customization

### Change Colors
Edit the CSS in `index.html`:
- Primary gradient: `background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);`
- Change `#667eea` and `#764ba2` to your brand colors

### Change Prices
- Edit the price in each package card
- Update the Square link amounts
- Update the JavaScript `buyPackage` function

### Add More Packages
- Copy a `.package` div
- Modify the content
- Add a new Square link to the `squareLinks` object
