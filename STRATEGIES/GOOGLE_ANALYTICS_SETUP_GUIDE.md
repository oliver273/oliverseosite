## üéØ SETUP PROCESS (For You)

1. **Ask client:** "Do you have a Google Analytics account?"
   - If YES: Ask for their tracking code
   - If NO: Guide them to create one (5 minutes, free)

2. **Get tracking code:**
   - Format: `G-XXXXXXXXXX` or `UA-XXXXXXXXX-X`
   - Client provides this (from their Google Analytics account)

3. **Add code to website:**
   ```html
   <!-- Google Analytics -->
   <script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
   <script>
     window.dataLayer = window.dataLayer || [];
     function gtag(){dataLayer.push(arguments);}
     gtag('js', new Date());
     gtag('config', 'G-XXXXXXXXXX');
   </script>
   ```
   - Add this before `</head>` tag
   - Replace `G-XXXXXXXXXX` with client's code

4. **Test it:**
   - Visit website
   - Check Google Analytics (should show 1 visitor = you)
   - Done!
