========================================
  QUICK START GUIDE
  OliverSEO Business Desktop App
========================================

STEP 1: Install Node.js
------------------------
If you don't have Node.js installed:
1. Go to: https://nodejs.org/
2. Download the LTS version
3. Install it (just click Next, Next, Next)

STEP 2: Run the App
-------------------
Double-click: START_APP.bat

That's it! The app will open in a window.

========================================
  OPTIONAL: Create Installer
========================================

If you want to create a proper installer (.exe file):

1. Double-click: INSTALL.bat
2. Wait for it to finish
3. Go to the "dist" folder
4. Run the .exe installer
5. Install the app like any other program

========================================
  KEYBOARD SHORTCUTS
========================================

Ctrl+D = Open Dashboard
Ctrl+R = Open Reddit Tool
Ctrl+Q = Quit App
F11 = Fullscreen

========================================
  TROUBLESHOOTING
========================================

Problem: "Node.js is not installed"
Solution: Install Node.js from nodejs.org

Problem: App won't start
Solution: 
1. Open Command Prompt in this folder
2. Type: npm install
3. Type: npm start

Problem: Reddit Post Finder not working
Solution: This is normal - use the direct links in Subreddits tab
