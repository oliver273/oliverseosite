========================================
  HOW TO USE YOUR LOGO
========================================

STEP 1: Save Your Logo
----------------------
Save your logo image file in the "OliverSEO_Site_Ready" folder with one of these names:
  - logo.png
  - logo.jpg
  - logo.jpeg
  - dashboard_logo.png
  - dashboard_logo.jpg

STEP 2: Convert to Icon
------------------------
Double-click: UPDATE_LOGO.bat

This will:
  1. Convert your logo to icon.ico
  2. Update the desktop shortcut with your logo

That's it! Your shortcut will now show your logo.

========================================
  MANUAL METHOD
========================================

If the batch file doesn't work:

1. Save your logo as logo.png in this folder
2. Open Command Prompt here
3. Run: python convert_logo_to_icon.py
4. Run: cscript create_shortcut.vbs
